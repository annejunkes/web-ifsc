Papel do HTTP

O HTTP (Hypertext Transfer Protocol) é o "idioma" que navegador e servidor usam para conversar. Ele define as regras de como pedir e enviar informações na web.

Cliente, Servidor, Requisição e Resposta

Cliente: quem faz a pergunta (ex: seu navegador).
Servidor: quem responde (ex: computador que guarda o site).
Requisição (Request): o pedido que o cliente faz ao servidor.
Resposta (Response): a resposta que o servidor devolve ao cliente.

Métodos GET e POST

GET: usado para pegar informações do servidor (ex: abrir uma página).
POST: usado para enviar informações ao servidor (ex: enviar um formulário).

Códigos de Status HTTP

200 OK – deu tudo certo.
301 Movido permanentemente – a página mudou de endereço.
400 Requisição inválida – o pedido está errado.
403 Proibido – você não tem permissão para acessar.
404 Não encontrado – a página não existe.
500 Erro interno do servidor – algo quebrou no servidor.
