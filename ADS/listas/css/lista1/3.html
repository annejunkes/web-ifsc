Formas de aplicar CSS

Existem 3 formas de aplicar estilo a página:

1. Inline: dentro da própria tag do elemento
Ex: <div style="background-color: purple">

2. Interno: dentro da head em tag style do html
Ex:
<head>
____<style>
____div{ background-color: yellow; }
____/<style>
</head>

3. Arquivo css: tag de link no head
Ex:
<head>
____<link rel="stylesheet" href="./css/style.css">
</head>
