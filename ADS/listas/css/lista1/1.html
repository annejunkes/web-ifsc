Regra CSS

Uma regra CSS (ou CSS rule) é a unidade fundamental de estilização em CSS. Ela é composta por:
Seletor → indica qual elemento HTML será estilizado
Bloco de declaração → contém as instruções de estilo, envolvidas por { }
Declarações → cada uma composta por uma propriedade e um valor, separados por :

seletor { propriedade: valor; }

h1 { color: blue; }
