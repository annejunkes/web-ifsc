Formas de aplicar CSS

Existem 3 formas de aplicar estilo a página:

1. Inline: dentro da própria tag do elemento
Ex: <div style="background-color: purple">

2. Interno: dentro da head em tag style do html
Ex:
<head>
____<style>
____div{ background-color: yellow; }
____/<style>
</head>

3. Arquivo css: tag de link no head
Ex:
<head>
____<link rel="stylesheet" href="./css/style.css">
</head>

Seletores

Básicos

*{} /* universal */
elemento{}
.classe{}
#id{}
seletor1, seletor2 {} /* agrupa */

Combinadores

div p {} /* descendente (qualquer nível) */
div > p {} /* filho direto */
h1 + p {} /* irmão adjacente (imediatamente após) */
h1 ~ p {} /* irmão geral (depois, mesmo nível) */

Box Model

*cada elemento é uma caixa
*margin (externo)
*padding (interno)
*border (necessário especificar tipo (solid, dotted, ..))
*width e height afetam apenas content por padrão

Positions

valor comportamento
static padrão - segue fluxo normal
relative desloca a partir de sua posição original (espaço original reservado)
absolute posiciona em relação ao ancestral posicionado mais próximo (pai com relative/absolute/fixed). Sai do fluxo.
fixed posiciona em relação à janela (viewport). Não rola.

Propriedades de deslocamento

top, right, bottom, left (funcionam apenas com position diferente de static)
