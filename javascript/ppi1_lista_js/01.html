O que é Javascript?

1 - Qual foi a motivação para a criação do JavaScript?
R: A falta de iteratividade da web, já que ela era construída apenas com HTML e CSS. Os usuários não podiam interagir com as páginas, tornando a experiência na web muito limitada. Com o objetivo de oferecer mais funcionalidade para as páginas surgiu o Javascript em 1995.

2 - Qual é a diferença entre execução no navegador e no servidor?
R: A execução no servidor necessita de um envio de requisição, para processar e recarregar toda a página, ou encaminharo usuário a outra página, reexecutando dezenas de requisições para download dos html, css, imagens.
Com a execução no navegador, as aplicações rodam em parte no cliente, de forma que somente quando há necessidade de algum dado, é requisitado apenas uma pequena porção de dados ao servidor. As vantagens de execução no navegador é a rapidez e fluidez no carregamento das páginas.

3 - Qual é conceito de sandbox no navegador?
R: sandbox é uma caixa dentro do navegador que roda código (encapsulada), envia requisições de forma limitada para o servidor, acessa alguns eventos, interage com o html da página. O Javascript roda dentro da sandbox Ele acessa dados em memória.

4 - O que são os objetos globais como document, console e alert?
R: Um objeto é uma ocorrência de um tipo personalizado para o programa ou contexto, é nada mais do que um conjunto de atributos aninhados a uma variável denomina-se um objeto.

---------document-----------------
É um objeto que representa o DOM (Document Object Model) da página. O DOM é uma representação hierárquica em forma de árvore de um documento HTML ou XML, que permite que o código JavaScript interaja com os elementos da página, manipulando-os e alterando sua aparência e comportamento.

---------console-------------------
É um objeto que representa o console JS, inclusive a sua sub função log() imprime no terminal e clear() imprime no terminal.

-----------alert----------------------
A função window.alert() exibe uma caixa de diálogo com uma mensagem para o usuário.
