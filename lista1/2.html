Textos

Esta é uma tag de parágrafo, ela é uma tag em bloco, ou seja, ocupa toda a largura da página

Esta é uma tag span, ela é uma tag inline, ou seja, ocupa apenas o espaço do seu conteúdo

Esta é a tag em, ela é uma tag que marca a importância semântica do conteúdo, deixa o texto por padrão em itálico

Esta é a tag b, ela é uma tag de estilização, deixa o conteúdo por padrão em negrito (bold)

Esta é a tag i, ela é uma tag de estilização, deixa o conteúdo por padrão em itálico

Esta é a tag mark, ela é uma tag de destaque de texto, deixa a marcação sobre o conteúdo em amarelo por padrão
Esta é a tag sub, ela é uma tag que altera a altura do texto, deixa o conteúdo alinhado um pouco abaixo do resto do texto

EX: H 2O

Esta é a tag sup, ela é uma tag que altera altura do texto, deixa o conteúdo alinhado um pouco acima do resto do texto

EX: E = mc2
