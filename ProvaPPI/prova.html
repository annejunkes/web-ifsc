R:1 - Ao pressionar Enter, será chamado o DNS(nome de domínio do site, exemplo: www.ifsc.com.br), o IP (endereço) será enviado para o link, será feita uma requisição HTTP, um aplicativo vai procurar essa requisição e formular uma resposta HTTP. Caso o redirecionamento para o link solicitado dê certo, será renderizada a estilização e lógica da página (css e javascript), o site irá abrir e a resposta no cabeçalho HTTP será OK, senão aparecerá uma página html informando qual tipo de resposta HTTP o link gerou (exemplo: Error 404)


R:2 - Não, HTML é uma linguagem de marcação de texto (Hyper Text Mark Language), ela utiliza tags para classificar os elementos na tela e não possui variáveis para armazenar informação. O CSS também não é uma linguagem de programação e tem finalidade de estilização de telas, sejam elas web ou desktop. O Javascript é uma linguagem de programação e é utilizado para adicionar funcionalidade às telas, como ações em botões e fluxo de telas


R:3 - Uma regra CSS padroniza a estilização de elementos da mesma classe, id ou tag.
Exemplo:
div{ => div é o seletor css, tag/classe/id que vai ser padronizada (nesse caso é a tag)
height: 20lvh=> height é um atributo da div que vai definir a altura dela;
width: 20lvw=> width é um atributo da div que vai definir a largura dela
}
Obs: Alguns seletores tem prioridade sobre outros, como o id sobre a classe e a classe sobre a tag
